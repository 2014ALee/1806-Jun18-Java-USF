What is CSS?

CSS allows you to create rules that specify how the conetn of an element should appear. For example. you can specify that the background of a page is a light grey, all the paragraphs should appear in dark grey and use teh Arial typeface, or that all level one headings should be in blue, italic, and Times typeface.


Understanding CSS: Thinking inside the box

The key to understanding how CSS works is to imagine that there is an invisible box around every HTML element. Css allows us to create rules taht control the way that each individual box (and its contenst) is presnted.
