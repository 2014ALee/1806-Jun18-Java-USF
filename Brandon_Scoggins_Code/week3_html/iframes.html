Iframes

An iframe is like a little window that has been pasted into your webpage. In this window, you can see this other page. Iframe is an abbreviation for inline frame.

One commone use of iframes is to embed a google map into a page. The content of the iframe can be any html page (either localy or external)

"

Anther common use for iframes is to embed vides into a page. Video content is often included to support the other content on the page.



Alternatively, we can use the ⟨video⟩ tag (HTML5 only to embed locally stored videos)
