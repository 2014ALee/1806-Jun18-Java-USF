Iframes

An iframe is like a little window that has been pasted into your web page. In this window, we can see the other page. The term iframe is an abbreviation for "inline frame".

One common use of iframes is to embed a Google Map into a page. The content of the iframe can be any HTML page (either located on the web server, or externally)



Another common use for iframes is to embed videos into a page. Video content is often included to support the other content on the page.



Alternatively, we can use the ⟨video⟩ tag (HTML5 only) to embed locally (maybe external?) stored videos.
