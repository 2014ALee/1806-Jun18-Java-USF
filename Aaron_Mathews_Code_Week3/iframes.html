Iframe (In-line Frame)

An iframe is like a window to another webpage.

One common use of iframes is to embed google map into a page. The content of iframe can be any html (local or server).



Another common use for the iframes is to embed videos into a page. Video content is often included to support the other content on the page.

Alternatively, we can use the &LeftAngBracket;video⟩ tag (HTML5) to embed locally stored videos.
